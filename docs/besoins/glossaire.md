# Glossaire

| Terme                     | Définition                                                                                                                                                           |
| ------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Planifium**             | Outil officiel de l’UdeM qui fournit les informations sur les cours, les horaires, les préalables et l’offre de cours par programme et par session.                 |
| **Avis étudiant**         | Appréciation ou retour d’expérience formulé par un·e étudiant·e sur un cours (contenu, charge de travail, difficulté, évaluations, etc.), partagé avec d’autres. Principalement sur Discord.   |
| **Résultats agrégés**     | Statistiques globales associées à un cours (par exemple moyenne générale, taux d’échec, nombre d’inscrits) utilisées pour analyser la réussite et la difficulté.    |
| **Serveur Discord étudiant** | Espace de discussion non officiel où les étudiant·e·s d’un programme (par ex. le DIRO) échangent des informations, des conseils et des avis sur les cours.       |
| **Profil étudiant**       | Ensemble des caractéristiques académiques et personnelles d’un·e étudiant·e (cheminement dans le programme, intérêts, contraintes d’horaire, objectifs d’études).   |
| **Fiche cours**           | Description officielle ou synthèse des informations liées à un cours : titre, code, crédits, contenu, préalables, horaire et parfois modalités d’évaluation.         |
| **Loi 25**                | Loi québécoise sur la protection des renseignements personnels, qui encadre la collecte, l’utilisation, la conservation et la diffusion des données des individus.   |
